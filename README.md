# Java Foundations

A collection of Java exercises and mini-projects created while strengthening my skills in object-oriented programming, collections, exception handling, file I/O, and application design.

## Mini-Projects

### Student Grade Tracker

A Java console application that:

- Manages multiple students
- Stores a variable number of grades
- Calculates individual and class averages
- Determines pass/fail status
- Validates user input
